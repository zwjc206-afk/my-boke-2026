# 个人IP网站维护说明

## 预览网站

方式一：双击当前文件夹里的：

```text
start-preview.command
```

浏览器打开：

```text
http://localhost:4173
```

方式二：在当前文件夹运行：

```bash
python3 -m http.server 4173
```

## 更新个人信息、文章、图片

主要改这个文件：

```text
content/site-data.json
```

- `proof`：首页数字背书
- `expertise`：专业能力
- `gallery`：形象图片
- `videos`：讲课视频
- `articles`：文章卡片
- `fallbackNews`：没有联网更新时显示的AI资讯

图片放这里：

```text
public/assets/images/
```

视频放这里：

```text
public/assets/videos/
```

## 新增讲课短视频

如果原视频很大，先压缩成网页版本：

```bash
scripts/compress-video.sh "/你的电脑/原视频.mp4" "public/assets/videos/compressed/视频文件名.m4v"
```

最简单的方式：

```bash
node scripts/add-video.mjs "/你的电脑/视频.mp4" "视频标题" "一句话介绍" "/你的电脑/封面图.png"
```

如果没有封面图，也可以只写前三项：

```bash
node scripts/add-video.mjs "/你的电脑/视频.mp4" "视频标题" "一句话介绍"
```

脚本会自动把视频复制进网站素材目录，并把它加入首页视频区。

## 每天更新全球AI资讯

运行：

```bash
node scripts/update-ai-news.mjs
```

它会从以下RSS源抓取最新AI资讯，并写入：

```text
content/news.generated.json
```

当前接入源：

- OpenAI News RSS: https://openai.com/news/rss.xml
- Google AI RSS: https://blog.google/technology/ai/rss/
- NVIDIA Blog RSS: https://blogs.nvidia.com/feed/
- Microsoft AI Blog RSS: https://www.microsoft.com/en-us/ai/blog/rss

页面会显示中文标题、中文摘要和资讯图片；如果原资讯源没有配图，会使用本地设计的 AI 资讯默认图。

## 更新合作企业 Logo

合作企业列表在：

```text
content/site-data.json
```

字段是 `partners`。如果有 Logo 图片，把图片放到：

```text
public/assets/logos/
```

然后在对应企业里增加：

```json
"logo": "public/assets/logos/文件名.png"
```

## 定时自动更新逻辑

也可以双击当前文件夹里的：

```text
update-ai-news.command
```

如果网站部署到服务器，建议每天早上 8 点执行：

```bash
cd "/Users/zwjc206/Documents/New project 2" && node scripts/update-ai-news.mjs
```

如果部署到 Vercel、Netlify 或其他静态托管平台，可以把这条命令放进平台的定时任务，更新后重新发布网站。

## 推荐素材规范

- 首页人物照：竖版，清晰，建议 1200px 以上宽度。
- 文章封面：横版，建议 1600 × 1000。
- 短视频：优先 MP4，H.264 编码，9:16 竖版。
- 单个公开视频建议压缩到 20-80MB，加载体验更稳定。
